# ARPS-2 Beginner Activities

These beginner programming activities are designed to introduce novices to
microcontroller programming using Arduino C/C++ running on the Arduino UNO R4
family of microcontrollers. These activities are created specifically for use
with the ARPS-2 circuit hardware -- feel free to modify them for use with other
hardware devices.

## Curriculum Overview

- the Microcontroller Core Curriculum file provides a complete curriculum outline
